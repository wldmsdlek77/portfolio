작업환경 종합관리 플랫폼

2023.09 - 2024.02 (인턴 프로젝트)

프로젝트 소개

작업장에서 발생하는 유해 화학물질을 센서를 통해 모니터링하고, 환기수준 점검 및 평가가 가능한 웹사이트 입니다.

사용 스킬

  • Front-End : JavaScript
  • Back-End : Java   Spring Framework   MyBatis   PostgreSQL
  • Tools : Eclipse   SVN   사내 HTML 개발 툴

주요 기능

  • 화학물질, 환기수준 상시 모니터링
  • 사업장 현장지원 신청
  • 센서장비 관리 및 시료 분석 신청
  • 공지사항, FAQ문의, 1:1 문의 등 고객지원

역할

  • 전체회원 정보 및 내정보 관리 기능 구현
  • 현장 지원 신청, 시료분석 신청, 공지사항, 1:1문의 등 게시판 기능 구현
  • PostgreSql와 Mybatis를 통한 CRUD, 페이지네이션, 파일 관리 쿼리 작성
  • SVN 연동을 통한 협업 및 소스코드 관리
