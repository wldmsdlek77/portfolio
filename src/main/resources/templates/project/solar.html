태양광 모니터링 시스템

2021.10 - 2021.12 (교내 프로젝트)

프로젝트 소개

태양광 발전기로부터 수집한 데이터를 분석 및 전처리 하여

현재출력, 발전량, 그래프등 요약된 모든 정보를 한 눈에 확인할 수 있는 모니터링 시스템 입니다.

사용 스킬

  • Front_End & Back-End : Python3
  • Tools : VSCode

주요 기능

  • 현재 출력량 및 선택날짜 출력량 확인
  • 금일 · 전일 발전량 수치 및 그래프 출력
  • 선택한 월의 주차별 발전량 그래프 출력

역할

  • Python의 pandas, numpy, matplotlib를 통한 csv 데이터 전처리
  • DataFrame 모듈을 통해 전처리된 데이터를 그래프로 시각화
  • Python의 tkinter library를 통해 화면 구현

메인 화면

메인화면 (날짜 선택시 데이터 출력)

주차별 발전량 그래프 (메인화면 상단의 보고서 탭 클릭시 출력)
