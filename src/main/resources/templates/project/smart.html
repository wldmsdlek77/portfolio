스마트팜 모니터링 시스템

2022.04 - 2022.06 (교내 프로젝트)

프로젝트 소개

코로나 규제 완화로, 집에 머무는 시간이 줄어 밖에서도 식물을 관리할 수 있는 시스템의 필요성을 절감하였습니다.

아두이노 센서로부터 온도, 습도, 토양 내수분, 조도 수치를 받아와 웹페이지에 출력하고

일정 수치가 넘을 경우 알림 메시지를 제공하는 웹사이트 입니다.

사용 스킬

  • Front_End : HTML   CSS   JavaScript
  • Back-End : Flask   MySQL
  • Tools : VSCode

주요 기능

  • 현재 온도, 습도, 토양 내 수분, 조도 수치 제공
  • 최적 수치 가이드 메시지 제공
  • 적정 수치를 벗어날 경우 알림 메시지 출력

역할

  • 모니터링 대시보드 화면 개발
  • 식물 관리에 최적화된 상태를 벗어날 경우 알림메시지 출력
  • 프로그레스바를 통한 센서 데이터 수치 시각화
  • 각 항목의 당일 수집 데이터를 그래프로 시각화

메인 화면

가이드 메시지 (최적 환경 설정을 도움)

당일 수집 데이터 그래프 (각 항목의 수치 클릭시 출력)
