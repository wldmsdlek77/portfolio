프로젝트 소개

기존 여행지 관련 어플 및 웹사이트는 불필요한 정보가 많이 노출된다는 문제가 있었습니다.

이를 개선하기 위해, 사용자가 원하는 여행 테마를 선택하면 그에 맞는 여행지 목록을 보여주는 앱을 개발했습니다.

또한 여행지 추천 리스트에서 관심 버튼을 클릭하면, 딥러닝 알고리즘을 통해 사용자의 선호도를 학습 하여

이를 바탕으로 사용자에게 맞춤형 여행지 추천을 제공합니다.

사용 스킬

  • Front-End : Flutter
  • Back-End : Python3   Flask   MariaDB
  • Tools : Android Studio   Jupyter Notebook   Git

주요 기능

  • 사용자의 관심 테마를 활용한 여행지 추천
  • 좋아요 수가 많은 여행지를 정렬하여 제공
  • 여행지 상세정보 제공 (테마, 위치 등)

역할

  • 인스타그램 게시물의 사진, 해시태그, 장소 등의 데이터를 수집하는 파싱 작업 진행
  • Python의 pandas 라이브러리를 활용한 데이터 라벨링 프로그램 개발
  • Flutter를 활용한 앱 메인화면, 관심 여행지 리스트, 내정보 화면 개발
  • 시작 화면
  • 회원가입 화면(여행테마 선택 가능)
  • 로그인 화면
  • 로그인 후 메인화면
  • 메인화면의 추천장소 선택시 상세정보 제공
  • 상세페이지의 위치확인하기 버튼클릭시 Map 제공
  • 메인화면에서 좋아요를 표시한 장소 목록
  • 프로필 화면
